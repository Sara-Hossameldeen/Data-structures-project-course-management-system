# Data-structures-project-course-management-system
A C++ console-based Course Management System that uses linked lists and file handling to manage students, courses, enrollments, and admin operations. The system supports registration, login, course enrollment, GPA updates, and transcript generation.

# Features
Student:
- Register and login
- Browse available courses
- Enroll in courses
- View transcript (grades & enrolled courses)
Admin:
- CRUD operations for students
- CRUD operations for courses
- Update student GPA
- View course reports (students + grades)
- Monitor enrollment statistics

# Data Structures Used
- Singly Linked Lists:
  Students
  Courses
  Enrollments
  Admins

# File Handling
students.txt ->	Student records
courses.txt	 -> Course details
enrollments.txt	-> Enrollment data
admin.txt	Admin -> credentials

# System Workflow
Start Program
   ↓
Load Data from Files
   ↓
Main Menu:
   1. Student Login
   2. Admin Login
   3. Register Student
   4. Exit
